# sqlonline
Qui potete trovare [il sito online](https://petrangolini.github.io/sqlonline/) di questo repository. Questo sito utilizza [sql.js](https://github.com/sql-js/sql.js/) per creare un database sqlite online front-end, e la libreria [faker.js](https://fakerjs.dev/) per popolare il database con dati realistici. Il sito ha lo scopo di creare in maniera molto rapida database sqlite per fini didattici, infatti si puo popolare il database con dati casuali e con dati specifici e si pu√≤ esportare il database e condividerlo con la classe per richiedere poi di fare le ricerche dei dati specifici.
